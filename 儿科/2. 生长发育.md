# 小儿生长发育的规律

## 一般规律
* 由上到下，先抬头、再抬胸、后会坐
* 由近到远，从手臂到手，腿到足
* 由粗到细，从抓到拾

## 各系统、器官生长发育不平衡
* 神经系统发育最早
* 生殖系统发育最晚
* 体格发育是快慢快
* 淋巴系统是先快后慢 

# 体格生长的常用指标
体重、身高、头围、胸围

## 体重
* 初生儿体重平均3KG
* 3个月约6KG(2倍)
* 1岁余额10KG(3倍)

* 公式
< 6月 体重 = 出生体重 + 月龄 x 0.7
7 ~ 12月  体重 = 6 + 月龄 x 0.25
2 ~ 12岁  体重 = 8 + 年龄 x  2

* 体重会出现暂时性生理性体重下降，下降范围为3% ~ 9%，如果体重下降超过10%，或10天未恢复，为病理性体重下降。

## 身高
* 身高反应骨骼发育的重要指标
* 出生时身长约50，1岁时75cm,2岁87cm,
* 2 ~ 12岁身高计算公式： 身高 = 75 + 年龄 x 7
